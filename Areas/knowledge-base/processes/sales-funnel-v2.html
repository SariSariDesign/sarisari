SKILL — Lead Connection HUMAN REVIEW

Lead Connection

  • Checks CRM for prior outreach to anyone at this company
  • Drafts first-touch message
  • Channel: LinkedIn first, email fallback
  • Tone: casual, not salesy

Contact: design lead if they have one, else whoever owns the hiring decision.

CHANGED
SKILL — Lead Nurture HUMAN REVIEW

Lead Nurture

  • Scheduled scan for overdue follow-ups
  • Checks for re-engagement signals (replies, activity)
  • Drafts follow-up using UX audit insights
  • LinkedIn engagement + DMs

Future: email campaigns, newsletter

CHANGED
DETECTION LAYER

Buy Signal → Call Booked

  • 3+ substantive replies in 2 weeks
  • Asked about services/pricing
  • Calendly booking (auto-advance)
  • New external signal (funding, hiring) while in nurture

Runs continuously during Nurture, not a separate stage.

SKILL — UX Audit

Website UX Audit

Auto-generated when lead status = "connected"

  • Checks if audit already exists before running
  • Report saved to Resources/[Company]/
  • Findings fuel nurture messages

Also feeds cross-industry pattern analysis over time.

NEW

Archived / Cold

Leads that exit the active pipeline:

  • No response after X touches across channels
  • Connected but zero engagement after follow-ups
  • No longer fits ICP
  • Explicit "not interested"

Not permanent — periodic re-evaluation on new signals. Soft pause option for warm-then-quiet leads.
